# IMC Prosperity 4

Repository for developing, testing, and submitting trading strategies for the IMC Prosperity 4 challenge.

## Structure

- `data/` raw and processed data
- `notebooks/` exploration and research
- `src/` reusable strategy and backtesting code
- `traders/` competition submission files
- `configs/` strategy parameters
- `logs/` experiment notes and results
- `tests/` test files
