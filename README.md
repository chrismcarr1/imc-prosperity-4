# IMC Prosperity 4

This repository contains my work for **IMC Prosperity 4**, IMC Trading's algorithmic trading competition. The project was built around researching market behavior, developing trading strategies, and organizing code for testing and submission.

## What I Built

- Strategy logic for the Prosperity trading framework
- Research notebooks for exploring market data and product behavior
- Reusable code for strategy development and backtesting
- Competition submission files organized under `traders/`
- Configuration files for strategy parameters and experiment setup
- Logs and notes for tracking strategy performance over time

## Focus Areas

The project centers on practical quantitative trading concepts, including:

- Fair value estimation
- Market making
- Price dislocation signals
- Inventory and position management
- Backtesting and iteration
- Risk-aware order generation
